# Palmabase
## Understanding your Palms

To checkout this project:

`git clone git@github.com:Mappingkat/Palmabase.git`

`cd Palmabase`

`bundle`

`rake db:setup`
`rake db:migrate`
`rake db:seed`

and finally...

`rails s`

*Project built on Rails 4.0.0, and uses Foundation 3*
